This page is designed to give you a overview about the website so that using the site is fun rather than being a head scratcher.

The basic purpose of this page is to share information and hence the name infoverse. We solve this purpose by providing our users with the ability to form groups and join them according to their interest.

Group :

A group has 4 major parts.

  • Title :

    A title is a name given to a group by its creator.

  • Description :

    A description gives a overview about the type of information that will be shared in the group.

  • Join :

    Every user of the site has the ability to join a group which means he/she will be able to see to all the information shared in that group as well as share relevent information by using the post feature.

  • Leave :

    If you have joined a group and now dont want to be the part of the group you have the option to leave the group. After leaving you cannot see the content shared in the group.

Member:

If a user has joined a particular group he/she will now be shown as a member of that group. More the members the popular the group. The more the posts the more active the group.

Post:

A post refers to sharing information in a group. In order to post, it is necessary for the user to join atleast one group. Every post will have a user's stamp who posted it and the date-time stamp.

By clicking on the user's stamp you can view all the posts the user has posted.
